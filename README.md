
# BraveHeart

BraveHeart is a 2D mobile platformer where speed is everything. You get to choose between 2 different characters, each with their own damage or health meaning different playstyles. Be ware or the knight pit, as defeating them is close to impossible. Your mission is to reach the final boss and defeat them in the shortest time possible.

## Features

- Dual Character System - Choose between two distinct characters, each offering a different gameplay experience

- Time Attack Gameplay - Every second counts as you sprint through levels to achieve the fastest completion time

- High Score System - Your best times are automatically saved, challenging you to beat your personal records

- Boss Battle - Face off against a challenging final boss that tests all your platforming skills

## Unity:
- Version: 2022.3.59f1
- Libraries:
    - InputSystem: Originally was made for playing with WASD instead of buttons
    - Cinemachine: for setting up the camera that follows the player
    - Rest Client for Unity: To connect the nodeJS backend with unity for login, signup and highscore

## backend:
- NodeJS: install using "npm install"
- ExpressJS: install using "npm install express"
- body-parser: install using "npm install body-parser"
- cors: install using "npm install cors"
